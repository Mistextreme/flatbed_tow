Professional Advanced Towing System (ESX-Legacy)Este sistema é uma solução completa para reboque de veículos em servidores de Roleplay, eliminando a necessidade de configurar coordenadas manualmente.FuncionalidadesModo Edição com Laser: Configure qualquer veículo in-game. O laser detecta bones, calcula offsets e gera a configuração pronta.Rampas Dinâmicas: Suporte para rampas nativas do modelo ou rampas físicas spawnadas (prop_roadcone02a ou rampas custom).Sistema de Winch (Guincho): Cabos físicos sincronizados entre todos os jogadores.Anexação Sincronizada: Veículos presos movem-se com a rampa e o camião sem "jittering".Integração ESX: Permissões por grupo e notificações integradas.InstalaçãoCopie a pasta advanced_tow para o seu diretório resources.Adicione ensure advanced_tow no seu server.cfg.Configure os grupos com acesso ao modo edição no config.lua.Comandos/towedit: Abre o modo de edição (Requer permissão)./deployramp: Estende/Recolhe a rampa do veículo./winch: Ativa o sistema de guincho no veículo mais próximo.Como configurar um novo veículoEntre num veículo de reboque.Use /towedit.Aponte o laser para o veículo e siga as instruções no ecrã para selecionar o Bone de ancoragem.Se necessário, use os controlos (Setas/PageUp/Down) para ajustar a posição da rampa.Use /saveconfig para exportar os valores para a consola/log.
